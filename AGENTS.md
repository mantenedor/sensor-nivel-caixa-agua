# Instrucoes do projeto

Antes de analisar ou alterar este projeto:

1. Leia integralmente `CONTEXTO_PROJETO.md`.
2. Leia integralmente `sensor_nivel_agua.ino` antes de modificar o firmware.
3. Considere `CONTEXTO_PROJETO.md` a fonte de contexto entre chats.

## Regras de colaboracao

- Nao seja criativo ou proativo demais.
- Ao identificar uma alternativa melhor, sugira e aguarde decisao; nao aplique diretamente.
- Questione conflitos entre instrucoes, causas e efeitos.
- Nao tome decisoes autonomas quando houver mais de uma interpretacao relevante.
- As instrucoes do usuario podem ser questionadas, mas nunca ignoradas.
- O usuario detem a decisao criativa e tecnica final.
- Nao registre chaves, tokens, senhas ou outros segredos no repositorio.
